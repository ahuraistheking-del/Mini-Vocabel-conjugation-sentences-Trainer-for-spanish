# 🇪🇸 AI-Powered Spanish Vocabulary Trainer

Ein smarter, KI-gestützter Spanisch-Vokabeltrainer, der mit Python und der OpenAI API entwickelt wurde. Dieses Projekt hilft dabei, sich perfekt auf das Leben und die Bürokratie in Spanien vorzubereiten.

## 🚀 Features

* **KI-Feedback (`gpt-4o-mini`):** Wenn du ein Wort falsch übersetzt, zeigt dir die KI nicht nur die Lösung, sondern erklärt deinen Fehler, gibt dir eine Eselsbrücke und erstellt einen Beispielsatz aus dem echten Leben in Spanien.
* **Tolerante Rechtschreibung:** Das Programm erkennt deine Antwort auch dann als richtig an, wenn du mal einen spanischen Akzent (`á`, `ó`, `ñ`) vergessen hast.
* **Sicherheit geht vor (Sixth):** Das Projekt wurde mit der **Sixth**-Erweiterung überprüft. Alle API-Schlüssel sind sicher in einer `.env`-Datei versteckt und werden durch die `.gitignore` niemals auf GitHub hochgeladen.

---

## 🛠️ Tech Stack

* **Sprache:** Python 3
* **KI-Anbindung:** OpenAI API (`openai`)
* **Sicherheit:** Python-dotenv & Sixth Vulnerability Scanner
